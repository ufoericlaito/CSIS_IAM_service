## 🎨 Container 组件演示

所有 Container 组件都在管理员首页有完整演示！

### TextContainer - 文字容器
```tsx
<TextContainer
  title="标题"
  content="内容文字或 JSX 元素"
  variant="highlight"  // default | highlight | info | warning
/>
```

### TableContainer - 表格容器
```tsx
<TableContainer
  title="表格标题"
  columns={[
    { key: 'id', label: 'ID' },
    { key: 'name', label: '姓名' },
  ]}
  data={数据数组}
  onRowClick={(row) => console.log(row)}
/>
```

### ImageContainer - 图片容器
```tsx
<ImageContainer
  title="图片标题"
  images={[
    { src: '图片URL', alt: '描述', caption: '说明文字' }
  ]}
  columns={3}  // 2 | 3 | 4
/>
```

### VideoContainer - 视频容器
```tsx
<VideoContainer
  title="视频标题"
  videos={[
    { src: '视频URL', title: '标题', description: '描述' }
  ]}
  columns={1}
/>
```
