# UL 学生管理系统 - 项目说明

## 项目概述

这是一个仿照 UFOERIC 个人网站风格设计的学生账户管理和展示系统。系统采用现代化的 React + TypeScript + Vite 技术栈，使用 Container 设计模式分别管理文字、表格、图片和视频内容。

## 核心特性

### 1. Container 设计模式
系统的核心是四个可复用的 Container 组件：

#### TextContainer（文字容器）
- **位置**: `src/Container/TextContainer.tsx`
- **功能**: 展示文字内容
- **特性**:
  - 支持标题和正文
  - 4种样式变体：default, highlight, info, warning
  - 玻璃态效果
  - 响应式布局

#### TableContainer（表格容器）
- **位置**: `src/Container/TableContainer.tsx`
- **功能**: 展示表格数据
- **特性**:
  - 自定义列配置
  - 支持自定义渲染函数
  - 行点击事件
  - 悬停高亮效果
  - 渐变色表头

#### ImageContainer（图片容器）
- **位置**: `src/Container/ImageContainer.tsx`
- **功能**: 展示图片内容
- **特性**:
  - 网格布局（2/3/4列可选）
  - 图片预览模态框
  - 图片说明文字
  - 悬停放大效果

#### VideoContainer（视频容器）
- **位置**: `src/Container/VideoContainer.tsx`
- **功能**: 展示视频内容
- **特性**:
  - 响应式视频播放器
  - 16:9 宽高比
  - 自动播放选项
  - 视频信息卡片

### 2. 用户角色系统

#### 管理员角色
- **登录账户**: admin@ul.com / admin123
- **功能页面**:
  - 管理员首页 (`/admin/dashboard`)
  - 学生管理 (`/admin/users`)
  - 审计日志 (`/admin/audit-logs`)
- **权限**:
  - 查看所有学生信息
  - 编辑学生信息
  - 删除学生
  - 查看系统统计
  - 查看操作日志

#### 学生角色
- **登录账户**: 任意邮箱和密码
- **功能页面**:
  - 学生首页 (`/user/dashboard`)
- **权限**:
  - 查看个人信息
  - 查看课程信息
  - 访问学习资源

### 3. UI 设计风格

#### UFOERIC 风格元素
- **渐变背景**: 135度线性渐变 (#667eea → #764ba2)
- **玻璃态效果**: backdrop-filter blur + 半透明背景
- **动画效果**: fadeIn 入场动画、hover 变换
- **配色方案**: 紫色系主题色
- **卡片设计**: 圆角、阴影、边框光晕

#### 响应式设计
- 移动端优先
- 断点: 768px (平板), 992px (桌面)
- 自适应网格布局
- 触摸友好的交互


## 开发建议

1. **添加新页面**: 在 `src/Pages/` 创建组件，在 `App.tsx` 添加路由
2. **使用 Container**: 优先使用现有 Container 组件保持风格一致
3. **状态管理**: 在 `Store.tsx` 添加新的 action 和 state
4. **样式定制**: 遵循现有的玻璃态和渐变设计模式

## 注意事项

- 所有 TypeScript 错误在安装依赖后会自动解决
- 开发服务器支持热更新，修改代码后自动刷新
- 建议使用 Chrome/Firefox/Edge 最新版本浏览器
- 移动端访问需要使用 `--host` 参数暴露网络地址
